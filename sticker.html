SUMMARY

I extremely enjoy gifts that are personal and customised, which thus led me to start drawing and designing stickers as gifts to people. This became a tradition when I give gifts, hence this is a collection of the stickers I have designed thus far. This gallery features gifts to various friends as well as some I made for fun.
